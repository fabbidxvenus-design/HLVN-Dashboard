# Mission

## Problem

Quản lý hệ thống OCR cho warehouse/logistics: quản lý users, xem lịch sử scan của tất cả nhân viên, phân quyền admin/manager/user, theo dõi sử dụng API và chi phí.

## Target Users

- **Quản lý kho**: Theo dõi hoạt động scan của nhân viên, xem báo cáo, quản lý quyền truy cập
- **IT Admin**: Quản lý hệ thống, cấu hình API keys, theo dõi chi phí, quản lý users

## Unique Value

- **Tích hợp chặt chẽ với OCR mobile app**: Đồng bộ dữ liệu real-time, cùng hệ thống auth
- **Role-Based Access Control (RBAC)**: Phân quyền linh hoạt (admin, manager, user), chỉ admin vào dashboard
- **Quản lý tập trung**: Một nơi quản lý tất cả users, lịch sử scan, analytics, API usage
- **Serverless backend**: AWS Lambda giảm chi phí vận hành, scale tự động

## Success Metrics

- Admin có thể tạo/quản lý users trong < 2 phút
- Xem lịch sử scan của tất cả users với filter/search
- Theo dõi API usage và chi phí theo thời gian thực
- Dashboard load time < 2 giây
- 100% uptime với serverless architecture
